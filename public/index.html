Welcome to the NumberFact API

NumberFact is a free API for retrieving information about numbers in the form of json documents. The API also allows sending post, put and delete requests for adding, editing and deleting data in the database.


Numbers are cool for many reasons. First, they are the building blocks of all mathematical concepts, from basic addition and subtraction to advanced calculus and statistics. This means that numbers are the foundation of many scientific and technological advancements in fields such as physics, engineering, and computer science.


Usage

Retrieves all the data currently stored

            
                
GET https://numberfact.azurewebsites.net/ShowAllFacts

Example response

            [
            

            {
            
            
            "id": 0,
            

            "number": "5357",
            

            "factMessage": "5357 is the smallest number that can not be formed using the digit 1 at most 27 times, together with the symbols +, –, × and ÷."
            

            }
            

            ]

Returns an array with id, number and the fact message

Retrieves a object by the id

            
                
GET https://numberfact.azurewebsites.net/GetFactById/0

Send post request to the API

            
                
POST https://numberfact.azurewebsites.net/AddFact

When sending a post request, be sure to specify the right id so data array stays organized. The API will notify if an id change was made.

Send a post request to the server to fetch a new instance

            
                
POST https://numberfact.azurewebsites.net/AddRandomFact

Update one object

            
                
PUT https://numberfact.azurewebsites.net/UpdateFact/0

The user can send an id to specify which object should be updated. The API will automatically change the id of the post request to match the input id.

Delete fact from the database

            
                
DELETE https://numberfact.azurewebsites.net/DeleteFactById/0
